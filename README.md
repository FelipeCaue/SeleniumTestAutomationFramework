# When I Work Test Automation Demo

Required: 

* Firefox installed

Steps to execute:

* Extract Zip to local drive

* Open: \packages\nunit_runnable\bin\nunit.exe

* Click File > Open project and open \WIWDemoTests\bin\Debug\WIWDemoTests.dll

* Select tests to run

* Click Run
